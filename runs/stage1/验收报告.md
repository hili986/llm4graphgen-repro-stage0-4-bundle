# Stage1 验收报告

## 1) 阶段目标（一句话）

完成不依赖真实 LLM 的最小闭环：Provider 抽象 + MockProvider + OpenAIProvider + 图解析器 + smoke CLI，并保证输入输出可追溯落盘。

## 2) 完成清单（逐条）

- 已实现 Provider 抽象：`src/llm4graphgen/providers/base.py`。
- 已实现 `MockProvider`：`src/llm4graphgen/providers/mock_provider.py`。
- 已实现 `OpenAIProvider`：`src/llm4graphgen/providers/openai_provider.py`（仅从环境变量读取 `OPENAI_API_KEY`）。
- 已实现图结构解析器：`src/llm4graphgen/parsers/graph_parser.py`，支持空格/换行/尾逗号/重复边容错，并给出中文失败原因。
- 已实现 Stage1 smoke CLI：`src/llm4graphgen/stage1_smoke.py`，闭环执行 `prompt -> provider -> parser`。
- 已实现 LLM I/O 落盘：`llm_io.jsonl` 至少包含 `prompt/model/temperature/raw_output/parsed_result/parse_failure_reason`。
- 已实现运行日志：每次 CLI 运行写 `runs/<run_id>/run.log`（中文）。
- 已新增单测：`tests/test_stage1_smoke.py`，覆盖 provider、parser 与 CLI 关键链路。
- 已通过测试：`python -X utf8 -m pytest -q` 结果 `7 passed`。

## 3) 复核方法（给出可复制命令）

